Filenames are labels chosen by humans.
File types are facts discovered by inspecting the bytes.
Those two things often match, but they do not have to.

This room is your first reminder that an extension can lie.
Ask Linux what `poster.jpg` really is, then pull the readable text out of it instead of trusting the name.

**Your task:**
Identify the real file type of `poster.jpg`, recover the answer hidden inside it, and verify it with:

```bash
./submit ANSWER
```
